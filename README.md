**Proxy format:**  
`http://user:pass@ip:port`

**Proxy format:**  
`query_id%3D......`

**How to get query data**

F12 > Start Bot > Network Tab > Name Column chose line "telegram-auth?invitation...." > Headers tab > Request URL > Copy data from "query_id%3D....."
